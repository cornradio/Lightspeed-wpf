## Lightspeed 按键指南

### 键盘

| 按键 | 功能 |
|---|---|
| Shift + ← | 切换到上一个文件夹 (0→9 循环) |
| Shift + → | 切换到下一个文件夹 (9→0 循环) |
| ← → ↑ ↓ | 导航文件列表/图标 |
| Enter | 打开选中项 |
| Delete | 删除选中项 |
| F2 | 重命名选中项 |
| Alt + S | 显示/隐藏窗口 (可在设置中修改) |
| Ctrl + , | 打开/关闭设置面板 |

### 手柄 (XInput)

| 按键 | 功能 |
|---|---|
| LB / RB | 切换文件夹 (0~9), 长按连续滚动 |
| D-pad / 左摇杆 | 导航文件, 上下跨行移动 |
| A | 打开选中项 |
| X | 打开右键菜单 (在图标位置弹出) |
| B | 关闭右键菜单 |
| Back (Select) | 打开/关闭设置面板 |
| Start | 在文件资源管理器中打开当前文件夹 |

#### 右键菜单内

| 按键 | 功能 |
|---|---|
| D-pad ↑↓ / 左摇杆 | 移动高亮, 长按连续 |
| A | 执行高亮项 |
| B 或 X | 关闭菜单 |

#### 手柄快捷键 (召唤/隐藏窗口)

在设置面板中可录入自定义组合键, 支持 LB/RB/LT/RT/A/B/X/Y/Start/Back 和方向键, 至少 3 键同时按下即可生效。快捷键在后台也能响应。
