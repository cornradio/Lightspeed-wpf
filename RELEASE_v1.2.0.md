## Lightspeed v1.2.0 Release Notes

### Gamepad Full Support

Complete XInput gamepad integration for couch/HTPC use:

- LB / RB switch between folders 0-9, long-press for continuous auto-scroll
- D-pad and left stick navigate file items; up/down moves across rows in icon view (not just sequentially)
- A button opens the selected file or folder
- X button opens context menu at the selected item's position (not at mouse cursor)
- Inside context menu: D-pad/stick to highlight items, A to execute, B or X to dismiss
- Navigation speed optimized for fast, responsive control with no restrictive delays

### UI Improvements

- Folder buttons 0-9 moved into the title bar with compact styling, saving vertical space
- Current folder name displayed in the bottom bar, visible even in fullscreen mode
- Bottom bar height reduced; all buttons (open, refresh, AHK, view toggle, settings) now use transparent icon-only style with subtle hover effects
- View toggle buttons use an underline accent for the active state

### Icon Quality

- Icons now sourced from the system Jumbo image list (256x256) for crisp display at all sizes
- Icon size slider now correctly resizes icons in both list and icon views
