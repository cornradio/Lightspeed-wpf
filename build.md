# Lightspeed-UI 项目分析文档

## 项目概述

**Lightspeed-UI** 是一个用 C# WinForms 编写的 Windows 快速启动器，配合 AutoHotkey (AHK) 实现快捷键功能。

### 主要功能
1. **快捷启动器** - 显示 0-9 十个文件夹，用 WebBrowser 展示内容
2. **自动隐藏** - 窗口贴在屏幕顶部，鼠标悬停时滑出显示
3. **AHK 脚本生成** - 自动生成快捷键脚本（Alt+0~9 打开文件夹等）
4. **系统托盘** - 后台运行，支持刷新/退出

## 技术栈

- .NET Framework 4.7.2 + WinForms
- WebBrowser 控件
- AutoHotkey 集成
- System Tray 托盘

## 主要组件

### 左侧按钮面板
- 前进/后退按钮 `<` `>`
- 0-9 数字按钮（用于切换文件夹）
- Settings 按钮（打开设置面板）

### WebBrowser 区域
- 显示 C:/lightspeed/{0-9} 文件夹内容
- 支持快捷键导航（Alt+左右方向键）

### 设置面板
- 创建文件夹
- 生成 AHK 脚本
- 启动 AHK
- 生成并启动
- AHK 自启动选项
- 启动时自动隐藏
- 透明度调节
- 下载依赖链接
- 帮助链接

### 系统托盘
- 刷新 AHK
- 退出程序

## 核心功能实现

### 自动隐藏动画
- 使用 Timer 检测鼠标位置
- 当鼠标接近屏幕顶部时触发滑入/滑出动画
- 500ms 延迟防止误触

### AHK 脚本生成
- 根据 C:/lightspeed/{0-9} 文件夹内容动态生成脚本
- 支持快捷方式首字母匹配启动
- 支持 Alt+方向键窗口导航

## UI 优化方向

1. **视觉设计** - 更现代的外观、颜色、字体
2. **布局优化** - 按钮面板、设置面板的布局调整
3. **交互体验** - 动画效果、悬停状态等
4. **细节打磨** - 图标、边框、圆角等
